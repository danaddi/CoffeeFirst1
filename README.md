# CoffeeFirst

Клиент-серверное приложения для автоматизации взаимодействия гостя и кофейни. Авторизация через Firebase Auth, работа с меню (фильтрация, корзина, заказы), бонусная система с QR-кодами, локация кофеен на карте через работу с API. Технологии: MVVM, Coroutines/StateFlow, Room, Firebase Realtime DB, Dagger Hilt, Navigation Component, SavedStateHandle для сохранения состояния. Асинхронные операции через Coroutines, кеширование данных в Room.
